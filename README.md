# Stoic Quotes

Welcome to the **Stoic Quotes** repo. This web app serves as a place for daily stoic insights, read stoic blogs, and shop for books.

## Features
- A curated collection of quotes from famous Stoic philosophers like Marcus Aurelius, Seneca, and Epictetus.
- User-friendly interface for easy browsing and searching of quotes.
- Responsive design to support both mobile and desktop views.
